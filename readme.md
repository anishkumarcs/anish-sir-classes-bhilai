# Anish Sir Classes — Static Education Platform (2026)

A **secure, minimal, production-ready static website** designed for structured academic and technical education pathways.

---

## Architecture Overview

This project follows a **pure static deployment model**:

- HTML5 + CSS3 only
- No JavaScript runtime dependencies
- No backend servers
- Immutable deployments via Netlify

The site is designed to be:
- Fast
- Secure
- SEO-friendly
- Print-safe
- Long-term maintainable
