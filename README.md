# Anish Sir Classes Website

A lightweight, secure, accessible, and SEO-friendly educational website built using modern HTML5, CSS3, and JavaScript.

## Overview

This project provides an online presence for Anish Sir Classes and includes information about:

- Educational Programs
- Student Development Initiatives
- Academic Guidance
- Contact Information
- Course Information

The website is designed with simplicity, performance, accessibility, and maintainability as primary goals.

---

## Technology Stack

### Frontend

- HTML5
- CSS3
- JavaScript (ES2026)

### Hosting

- Netlify

### Architecture

- Static Site Architecture
- Progressive Enhancement
- Mobile-First Design


## Production Checklist

Before deployment:

- Update contact information
- Update email address
- Update phone number
- Verify sitemap URLs
- Verify canonical URLs
- Replace favicon
- Validate HTML
- Test mobile responsiveness
- Test accessibility
- Verify Netlify form submission
